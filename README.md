# retro-40

A Fantasy Computer using Forth as its system language.

## About

## Inspiration
    Lambda/8
    Jupiter ACE

## Specs
    256x192 screen
    16-colour palettes

## Usage
    Forth is both the programming and the command language for
    Retro-40. You can type commands or even new definitions on the
    console.
